# Emotional diary
Демонстрация работы в гите ЯЛ 2025




# Основные функциональные требования

### 1) Пользователь может создавать несколько записей о настроении в течение дня

### 2) Оценка настроения и состояния
       Программа будет последовательно собирать информацию о настроении пользователя через ~5 простых окон:
       Окно ввода даты и времени
       Окно выбора настроения: 5 смайликов (радостный, злой, нейтральный, грустный, уставший)
       Окно причины настроения(пользователь вводит в текстовое поле свою причину выбранного настроения или "кратко о сегодняшнем дне"
       Окно комментария(текстовое поле для подробного описания чувств)
       Окно оценки дня: от 1 до 10

### 3)  Система учета всех оценок за неделю (В конце каждого неделю программа проанализирует все записи и даст полезные выводы)
        
        Сколько раз пользователь испытывал различные эмоции
        Сколько всего эмоций испытал пользователь
        Выдает советы на основе преобладающего настроения
        Рассчитывает среднюю оценку недели
        
        <u>Принцип расчета:</u>
                         Общее количество записей = 3 + 2 + 1 + 2 + 0 + 3 + 1 = 12 записей
                         Сумма всех оценок = 8+6+7 + 5+9 + 4 + 7+8 + 6+6+9 + 7 = 82
                         Средняя оценка недели = 82 / 12 = 6.83

### 4) Система советов (Программа будет давать советы для улучшения настроения)
       Если больше радостных дней, то (например): "Продолжайте в том же духе! Вы большой молодец"
       Если больше грустных дней, (например): "Погуляйте на свежем воздухе", "Пообщайтесь с друзьями", "Побалуйте себя чем-то вкусным"
       Если больше уставших дней,  советы:  "Выспитесь", "Отвлекитесь от дел на час"
       Если больше гневвных дней, то: "Побейте подушку", "Сходите в спортзал"
       Если больше нейтральных дней, то: "Испытайте новые эмоции! Сходите в музей или на шоппинг"

Все данные будут сохраняться и их можно будет экспортировать
Автоматическое сохранение в базу данных
Экспорт недельной статистики в TXT файл
Резервное копирование данных(?)

# Архитектура и дизайн
### Основные модули
    MainWindow: Меню и навигация
    WizardManager: Управляет пошаговым процессом ввода данных (настроение,  причина, комментарий, оценка)
    DatabaseManager: Отвечает за сохранение и загрузку всех записей
    StatsCalculator: Анализирует данные и выдает статистику за неделю
    AdviceGenerato): Выдает персонализированные советы на основе данных
    ExportManager: Сохраняет данные в TXT файлы

### Структура базы данных
    Количество таблиц: 2 таблицы(Запись настроения(дата записи, выбранное настроение (смайлик), причина настроения,  комментарий пользователя, оценка дня от 1 до 10) и Советы)

### Структура меню и панели инструментов
    Новая запись
    История записей
    Статистика за неделю

### Горячие клавиши
    Ctrl + N - Новая запись

    Ctrl + E - Экспорт статистики

    Ctrl + H - История записей
